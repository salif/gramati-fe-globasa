格洛巴萨语法

常用短语和表达

问候语

salom - 你好

xanti - 你好 ("和平")

bonsoba - 早上好

bonnuru - 日安/下午好

bonaxam - 晚上好

bonnoce - 晚上好 ("晚安")

告别语

weda - 再见

xanti - 再见 ("和平")

finfe (rioko) - 回见

finfe xaya - 待会儿见

finfe ner xaya - 一会儿见/很快见

bon soba - 早上好

bon nuru - 日安/下午好

bon axam - 晚上好

bon noce - 晚安

其他祝福语

注意：如上所示，带有 bon (好) 的问候语表达为复合词，而告别语表达为两个词的短语。 在下面的表达中，当说话者和听话者保持在一起时，使用复合词。 因此，两个词根在一起，组合成一个词。 相反，如果说话者或听话者离开，则使用两个词的短语。 因此，词根是分开的。

bonata 或 bon ata - 欢迎

bonxanse 或 bon xanse - 祝你好运

bonyam 或 bon yam - 祝你胃口好 ("好餐")

bonglu 或 bon glu - 干杯 ("好饮料")

bonturi 或 bon turi - 一路顺风，旅途平安

常见问题/回答

  • Yu sen kepul? 或 Yu kepul?
    你好吗？

Daymo bon, ji yu?
很好，你呢？

semibon
还不错

semi semi
马马虎虎

  • Yu name keto? 或 Yusu name sen keto?
    你叫什么名字？

Mi name... 或 Misu name sen...
我叫...

(To sen) yukwe, na xorkone yu. 或 Yukwe.
很高兴认识你。 或 令人愉快。

(To sen) furaha, na xorkone yu. 或 Furaha.
很高兴见到你. 或 很荣幸。

(Mi sen) hox na xorkone yu. 或 Mi sen hox.
(我)很高兴见到你。 或 我很高兴。

  • Yu ogar keloka?
    你住在哪里？

Mi ogar in...
我住在...

  • Yu sen of keloka?
    你是哪里人？

Mi sen of...
我来自...

  • Yu sen kemo lao? 或 Yu sen fe ke nyan?
    你多大了？

Mi sen lao fe... (nyan). 或 Mi sen fe... (nyan).
我...岁。

  • Kam yu (pala) Globasa?
    你会说格洛巴萨语吗？

Si, xosu.
是的，会一点。

  • Yu pala ke basa?
    你会说什么语言？

Mi pala...
我会说...

  • Kam yu aham?
    你明白吗？

(Si,) mi aham.
(是的，) 我明白。

(No,) mi no aham.
(不，) 我不明白。

  • Ren loga... kemaner (in Globasa)?
    ...用格洛巴萨语怎么说？

In Globasa, ren loga...
(在格洛巴萨语中，) 你说...

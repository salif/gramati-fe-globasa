<h2>その他の挨拶</h2>
<p>注：上記のように、<strong>bon</strong> (良い、うまく)
	を含む挨拶は複合語として表現されますが、別れの挨拶は2語のフレーズとして表現されます。以下の表現では、話し手と聞き手が一緒にいる場合は複合語が使われます。したがって、2つの語根は一緒に組み合わされて1つの単語になります。対照的に、話し手または聞き手のいずれかが立ち去る場合は、2語のフレーズが使用されます。したがって、語根は離れています。
</p>
<p><strong>bonata</strong> または <strong>bon ata</strong> - ようこそ </p>

<ul>
	<li><strong>Yu sen kepul?</strong> または <strong>Yu kepul?</strong><br /> お元気ですか？ </li>
</ul>

<p><strong>semibon</strong><br /> 悪くないです</p>
<p><strong>semi semi</strong><br /> まあまあです</p>
<ul>
	<li><strong>Yu name keto?</strong> または <strong>Yusu name sen keto?</strong><br /> あなたの名前は何ですか？</li>
</ul>
<p><strong>Mi name...</strong> または <strong>Misu name sen...</strong><br /> 私の名前は…です</p>
<p><strong>(To sen) yukwe, na xorkone yu.</strong> または <strong>Yukwe.</strong><br /> お会いできて嬉しいです。 または 嬉しいです。</p>
<p><strong>(To sen) furaha, na xorkone yu.</strong> または <strong>Furaha.</strong><br /> お会いできて光栄です。 または 光栄です。</p>
<p><strong>(Mi sen) hox na xorkone yu.</strong> または <strong>Mi sen hox.</strong><br /> お会いできて嬉しいです。 または 嬉しいです。</p>
<ul>
	<li><strong>Yu ogar keloka?</strong><br /> どこに住んでいますか？</li>
</ul>
<p><strong>Mi ogar in...</strong><br /> 私は…に住んでいます</p>
<ul>
	<li><strong>Yu sen of keloka?</strong><br /> 出身はどこですか？</li>
</ul>
<p><strong>Mi sen of...</strong><br /> 私は…の出身です</p>
<ul>
	<li><strong>Yu sen kemo lao?</strong> または <strong>Yu sen fe ke nyan?</strong><br /> あなたは何歳ですか？ </li>
</ul>
<p><strong>Mi sen lao fe... (nyan).</strong> または <strong>Mi sen fe... (nyan).</strong><br /> 私は…歳です。</p>
<ul>
	<li><strong>Kam yu (pala) Globasa?</strong><br /> グロバサを話しますか？</li>
</ul>
<p><strong>Si, xosu.</strong><br /> はい、少し。</p>
<ul>
	<li><strong>Yu pala ke basa?</strong><br /> 何語を話しますか？ </li>
</ul>
<p><strong>Mi pala...</strong><br /> 私は…を話します</p>
<ul>
	<li><strong>Kam yu aham?</strong><br /> わかりますか？</li>
</ul>

<p><strong>(No,) mi no aham.</strong><br /> (いいえ、)わかりません。</p>

</ul>

<h2>その他のあいさつ</h2>
<p>注：上記のように、<strong>bon</strong>（良い、うまく）を使ったあいさつは複合語で表現されますが、別れのあいさつは2語のフレーズで表現されます。以下の表現では、話し手と聞き手が一緒にいる場合は複合語が使用されます。したがって、2つの語根は一緒に組み合わされて1つの単語になります。対照的に、話し手または聞き手のいずれかが離れる場合は、2語のフレーズが使用されます。したがって、語根は離れています。
</p>
<p><strong>bonata</strong> <em>または</em> <strong>bon ata</strong> - ようこそ </p>
<p><strong>bonxanse</strong> <em>または</em> <strong>bon xanse</strong> - 幸運を祈ります </p>

<h2>よくある質問と回答</h2>
<ul>
	<li><strong>Yu sen kepul?</strong> <em>または</em> <strong>Yu kepul?</strong><br /> お元気ですか？ </li>
</ul>
<p><strong>Daymo bon, ji yu?</strong><br /> とても元気です、あなたは？</p>
<p><strong>semibon</strong><br /> まあまあです</p>
<p><strong>semi semi</strong><br /> そこそこです</p>
<ul>
	<li><strong>Yu name keto?</strong> <em>または</em> <strong>Yusu name sen keto?</strong><br /> あなたの名前は何ですか？</li>
</ul>
<p><strong>Mi name...</strong> <em>または</em> <strong>Misu name sen...</strong><br /> 私の名前は...です</p>
<p><strong>(To sen) yukwe, na xorkone yu.</strong> <em>または</em> <strong>Yukwe.</strong><br /> お会いできてうれしいです。または、うれしいです。
</p>
<p><strong>(To sen) furaha, na xorkone yu.</strong> <em>または</em> <strong>Furaha.</strong><br /> お会いできて光栄です。または、光栄です。</p>
<p><strong>(Mi sen) hox na xorkone yu.</strong> <em>または</em> <strong>Mi sen hox.</strong><br /> お会いできてうれしいです。または、うれしいです。
</p>
<ul>
	<li><strong>Yu ogar keloka?</strong><br /> あなたはどこに住んでいますか？</li>
</ul>
<p><strong>Mi ogar in...</strong><br /> 私は...に住んでいます</p>
<ul>
	<li><strong>Yu sen of keloka?</strong><br /> あなたはどこの出身ですか？</li>
</ul>
<p><strong>Mi sen of...</strong><br /> 私は...の出身です</p>
<ul>
	<li><strong>Yu sen kemo lao?</strong> <em>または</em> <strong>Yu sen fe ke nyan?</strong><br /> あなたは何歳ですか？ </li>
</ul>
<p><strong>Mi sen lao fe... (nyan).</strong> <em>または</em> <strong>Mi sen fe... (nyan).</strong><br /> 私は...歳です。 </p>
<ul>
	<li><strong>Kam yu (pala) グロバサ?</strong><br /> あなたはグロバサを話しますか？</li>
</ul>
<p><strong>Si, xosu.</strong><br /> はい、少し（だけ）。</p>
<ul>
	<li><strong>Yu pala ke basa?</strong><br /> あなたは何語を話しますか？ </li>
</ul>
<p><strong>Mi pala...</strong><br /> 私は...を話します</p>
<ul>
	<li><strong>Kam yu aham?</strong><br /> わかりますか？</li>
</ul>
<p><strong>(Si,) mi aham.</strong><br /> （はい、）わかります。 </p>
<p><strong>(No,) mi no aham.</strong><br /> （いいえ、）わかりません。</p>
<ul>
	<li><strong>Ren loga... kemaner (in グロバサ)?</strong><br /> （グロバサでは）...を何と言いますか？</li>
</ul>
<p><strong>In グロバサ, ren loga...</strong><br /> （グロバサでは）...と言います。 </p>

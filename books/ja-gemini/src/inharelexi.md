<h1>内容語：名詞、動詞、形容詞と副詞</h1>
<p>
</p>
<h2>名詞</h2>
<p>グロバサの名詞は、単数形と複数形を区別しません。</p>
<ul>
	<li><strong>maux</strong> - ネズミ、ネズミたち</li>
	<li><strong>kalamu</strong> - ペン</li>
</ul>
<p>グロバサの名詞には、定冠詞も不定冠詞もありません。</p>
<ul>
	<li><strong>janela</strong> - (一つの)窓、(その)窓</li>
</ul>
<p>もし明確さを強調する必要がある場合は、<strong>hin</strong> (これ/これらの) または <strong>den</strong> (あれ/それらの) を使用できます。</p>
<ul>
	<li><strong>hin kitabu</strong> - この本、これらの本、(その)本</li>
	<li><strong>den flura</strong> - あの花、それらの花、(その)花</li>
</ul>
<p>もし単数形を強調する必要がある場合は、<strong>un</strong> (一つ) を使用できます。</p>
<ul>
	<li><strong>un denta</strong> - 一本の歯、(一つの)歯</li>
	<li><strong>hin un denta</strong> - この(一本の)歯、(その)歯</li>
</ul>
<p>もし複数形を強調する必要がある場合は、<strong>plu</strong> (複数の) を使用できます。</p>
<ul>
	<li><strong>plu pingo</strong> - (複数の)リンゴ</li>
	<li><strong>den plu pingo</strong> - それらの(複数の)リンゴ、(その)リンゴ</li>
</ul>
<h3>性別</h3>
<p>グロバサでは、人や動物を表す名詞は通常、性別中立です。</p>
<ul>
	<li><strong>ixu</strong> - 大人(男性、女性)</li>
	<li><strong>nini</strong> - 子供(男の子、女の子)</li>
	<li><strong>gami</strong> - 配偶者(夫、妻)</li>
	<li><strong>mumu</strong> - 家畜(雄牛、雌牛)</li>
</ul>
<p>もし性別を強調する必要がある場合は、形容詞 <strong>fem</strong> (女性の) と <strong>man</strong> (男性の) を接頭辞として使用できます。</p>
<ul>
	<li><strong>femnini</strong> - 女の子; <strong>mannini</strong> - 男の子</li>
	<li><strong>femixu</strong> - 女性; <strong>manixu</strong> - 男性</li>
	<li><strong>femgami</strong> - 妻; <strong>mangami</strong> - 夫</li>
	<li><strong>femmumu</strong> - 雌牛; <strong>manmumu</strong> - 雄牛</li>
</ul>
<p><em><strong>fem</strong> の語源: 英語 (feminine)、フランス語 (féminin)、ドイツ語 (feminin)、スペイン語 (femenina)</em></p>
<p><em><strong>man</strong> の語源: 中国語 (男 “nán”)、フランス語 (masculin)、スペイン語 (masculino)、英語 (masculine)、ドイツ語 (männlich)、ヒンディー語
		(मर्दाना “mardana”)、ペルシア語 (مردانه “mardane”)</em></p>
<p>人を表す名詞のうち、わずかな数は性別を示します。</p>
<ul>
	<li><strong>matre</strong> または <strong>mama</strong> - 母またはママ</li>
	<li><strong>patre</strong> または <strong>papa</strong> - 父またはパパ</li>
</ul>
<p><em>注：親(両親)を表す性別中立の言葉は <strong>atre</strong> です。ママ/パパを表す性別中立の言葉は <strong>mapa</strong> です。</em></p>
<h3>文頭の句における名詞</h3>
<p><strong>Fe</strong> は、名詞を伴う文頭の句でよく使用されます。</p>
<ul>
	<li><strong>Fe fato,</strong> - 実際、実は</li>
	<li><strong>Fe fini,</strong> - 最後に</li>
	<li><strong>Fe bonxanse,</strong> - 幸運にも、都合良く</li>
	<li><strong>Fe asif,</strong> - 残念ながら、不運にも</li>
	<li><strong>Fe onxala,</strong> - 願わくば</li>
	<li><strong>Fe folo,</strong> - それゆえに、結果として、だから</li>
	<li><strong>Fe misal,</strong> - 例えば</li>
	<li><strong>Fe xugwan,</strong> - 通常</li>
	<li><strong>Fe benji,</strong> - 要するに、基本的に</li>
	<li><strong>Fe moy kaso,</strong> - いずれにせよ、とにかく</li>
	<li><strong>Fe alo kaso,</strong> - そうでなければ</li>
	<li><strong>Fe nunya,</strong> - 現在、今</li>
	<li><strong>Fe leya,</strong> - 過去に、以前に、かつて</li>
	<li><strong>Fe xaya,</strong> - 将来に、後で</li>
</ul>
<h3>同格</h3>
<p>グロバサでは、2番目の名詞が最初の名詞の同一性を特定する場合、前置詞を使用せずに名詞の後に別の名詞を続けることができます。これは<em>同格</em>として知られています。</p>
<ul>
	<li><strong>Hotel Kaliforni</strong> - ホテルカリフォルニア</li>
	<li><strong>Estato Florida</strong> - フロリダ州</li>
	<li><strong>Towa Babel</strong> - バベルの塔</li>
	<li><strong>Dolo Onxala</strong> - 希望通り</li>
	<li><strong>Myaw Felix</strong> - 猫のフェリックス</li>
	<li><strong>misu doste Mark</strong> - 私の友人マーク</li>
	<li><strong>lexi <em>kursi</em></strong> - 単語 <em>椅子</em></li>
</ul>
<h3>小辞 <em>di</em>：文化固有の言葉と固有名詞</h3>
<p>小辞 <strong>di</strong> は、グロバサで既に確立されている普通の言葉と<em>同一の形</em>を持つ文化固有の言葉や固有名詞をマークするために、オプションで使用できます。</p>
<ul>
	<li><strong>soho</strong> - 相互の、お互いの <ul>
			<li><strong>(di) Soho</strong> - ソーホー (ニューヨーク市の地区)</li>
		</ul>
	</li>
</ul>
<h3>小辞 <em>ci</em>：親愛と愛情</h3>
<p>親愛の情や愛情を示すために、名詞や固有名詞の後に小辞 <strong>ci</strong> を続けることができます。</p>
<ul>
	<li><strong>mama</strong> - ママ <ul>
			<li><strong>mama ci</strong> - マミー</li>
		</ul>
	</li>
	<li><strong>nini</strong> - 子供 <ul>
			<li><strong>nini ci</strong> - キッド</li>
		</ul>
	</li>
	<li><strong>Jon</strong> - ジョン <ul>
			<li><strong>Jon ci</strong> - ジョニー</li>
		</ul>
	</li>
</ul>
<h3>敬称：<em>Gao</em> と <em>Kef</em></h3>
<p>形容詞 <strong>gao</strong> (高い、背の高い) と名詞 <strong>kef</strong> (ボス、長) は敬称として使用できます。</p>
<ul>
	<li><strong>alimyen</strong> - 教師 <ul>
			<li><strong>gao alimyen</strong> - 先生</li>
		</ul>
	</li>
	<li><strong>papa</strong> - パパ <ul>
			<li><strong>kef papa</strong> - ボス</li>
		</ul>
	</li>
</ul>
<h3>名詞/動詞</h3>
<p>グロバサでは、名詞/動詞は名詞または動詞のいずれかとして機能できる言葉です。</p>
<ul>
	<li><strong>ergo</strong> - 仕事 (<em>名詞または動詞</em>)</li>
	<li><strong>danse</strong> - ダンス (<em>名詞または動詞</em>)</li>
	<li><strong>yam</strong> - 食事 (<em>名詞</em>) または食べる (<em>動詞</em>)</li>
	<li><strong>lala</strong> - 歌 (<em>名詞</em>) または歌う (<em>動詞</em>)</li>
</ul>
<h3>名詞/動詞の比較</h3>
<p>名詞/動詞の比較は、<strong>max</strong> (<em>より多く</em>)、<strong>min</strong> (<em>より少なく</em>)、<strong>dennumer</strong>
	(<em>その数の、同じくらい多く</em>)、<strong>denkwanti</strong> (<em>その量、同じくらい多く</em>)、<strong>kom</strong>
	(<em>として、より</em>)という言葉を使って、以下のように表現されます。</p>
<p>名詞の場合：</p>
<ul>
	<li><strong>max... kom...</strong> - ...より多い...</li>
	<li><strong>min... kom...</strong> - ...より少ない...</li>
</ul>
<p><strong>Mi hare max kitabu kom yu.</strong><br /> 私はあなたよりも多くの本を持っています。</p>
<p><strong>Yu hare min kitabu kom mi.</strong><br /> あなたは私よりも少ない本を持っています。</p>
<ul>
	<li><strong>max te/to kom...</strong> - ...より(それらの)多く</li>
	<li><strong>min te/to kom...</strong> - ...より(それらの)少なく</li>
</ul>
<p><strong>Mi hare max to kom yu.</strong><br /> 私はあなたより(それらを)多く持っています。</p>
<p><strong>Yu hare min to kom mi.</strong><br /> あなたは私より(それらを)少なく持っています。</p>
<ul>
	<li><strong>max kom</strong> - ...よりも多く</li>
	<li><strong>min kom</strong> - ...よりも少なく</li>
</ul>
<p><strong>Mi hare max kom cen kitabu.</strong><br /> 私は100冊以上の本を持っています。</p>
<p><strong>Yu hare min kom cen kitabu.</strong><br /> あなたは100冊未満の本を持っています。</p>
<ul>
	<li><strong>dennumer... kom...</strong> - ...と同じくらい多くの...</li>
</ul>
<p><strong>Te hare dennumer kitabu kom mi.</strong><br /> 彼女は私と同じくらい多くの本を持っています。</p>
<ul>
	<li><strong>dennumer te/to kom...</strong> - ...と同じくらい(それらの)多く</li>
</ul>
<p><strong>Te hare dennumer to kom mi.</strong><br /> 彼女は私と同じくらい多く持っています。</p>
<ul>
	<li><strong>denkwanti... kom...</strong> - ...と同じくらい多くの...</li>
</ul>
<p><strong>Yu yam denkwanti risi kom mi.</strong><br /> あなたは私と同じくらい多くの米を食べます。</p>
<ul>
	<li><strong>denkwanti to kom...</strong> - ...と同じくらい(それ)多く</li>
</ul>
<p><strong>Yu yam denkwanti to kom mi.</strong><br /> あなたは私と同じくらい(それ)多く食べます。</p>
<p>動詞の場合：</p>
<ul>
	<li><strong>max... kom....</strong> または <strong>max kom...</strong> - ...よりも多く</li>
</ul>
<p><strong>Myaw max somno kom bwaw.</strong><br /> または: <strong>Myaw somno max kom bwaw.</strong><br /> 猫は犬よりも多く眠ります。
</p>
<ul>
	<li><strong>min... kom....</strong> または <strong>min kom...</strong> - ...よりも少なく</li>
</ul>
<p><strong>Bwaw min somno kom myaw.</strong><br /> または: <strong>Bwaw somno min kom myaw.</strong><br /> 犬は猫よりも少なく眠ります。
</p>
<ul>
	<li><strong>denkwanti... kom...</strong> または <strong>denkwanti kom...</strong> - ...と同じくらい多く</li>
</ul>
<p><strong>Bebe denkwanti somno kom myaw.</strong><br /> または: <strong>Bebe somno denkwanti kom myaw.</strong><br />
	赤ちゃんは猫と同じくらい多く眠ります。</p>
<p><em>...すればするほど、...になる</em>を表現するために、グロバサは <strong>folki... max/min, max/min</strong> を使用します。</p>
<p><strong>Folki mi max doxo, mi max jixi.</strong><br /> 読めば読むほど、私はより多くを知ります。</p>
<p>これらの句の順序は入れ替えることができます。</p>
<p><strong>Mi max jixi, folki mi max doxo.</strong><br /> 私は読めば読むほど、より多くを知ります。</p>
<h3>動詞のカテゴリー <span id="falelexili_klase"></span></h3>
<p>動詞は辞書で<em>補助</em>、<em>コピュラ</em>、<em>他動詞</em>、<em>自動詞</em>、または<em>両用動詞</em>として定義されています。このセクションで見られる接尾辞（<strong>-cu</strong>、
	<strong>-gi</strong>、<strong>-ne</strong>、<strong>-do</strong>、<strong>-pul</strong>）は、以下の<a
		href="./inharelexi.html#pimpan_fikso"><em>一般的な接辞</em></a>で説明されています。接頭辞<strong>xor-</strong>については、<a
		href="./lexikostrui.html#lefefikso"><em>語形成：接頭辞</em></a>で説明されています。</p>
<h4>助動詞</h4>
<p><em>助動詞</em>は、直後に別の動詞が続きます。この別の動詞は省略されることもあります。グロバサには<em>助動詞</em>が3つしかありません。：<strong>abil</strong>（できる、能力がある）、<strong>ingay</strong>（すべきである、義務がある）、<strong>musi</strong>（しなければならない、する必要がある）。
</p>
<h4>コピュラ動詞</h4>
<p><em>コピュラ動詞</em>は、主語とその補語を結び付けます。現在、12の<em>コピュラ動詞</em>があります。：<strong>sen</strong>（である）、<strong>xorsen</strong>（になる）、<strong>sencu</strong>（になる）、<strong>sengi</strong>
	（にする）、<strong>kwasisen</strong>（のように見える）、<strong>okocu</strong>（のように見える）、<strong>orecu</strong>（のように聞こえる）、
	<strong>nasacu</strong>（の匂いがする）、<strong>xetocu</strong>（の味がする）、<strong>pifucu</strong>（触れた感じがする）、
	<strong>hisicu</strong>（身体的または感情的に感じる）、<strong>ganjoncu</strong>（感情的に感じる）。</p>
<h4>他動詞</h4>
<p><em>他動詞</em>は直接目的語を取ります。：<strong>haja</strong>（必要とする）、<strong>bujo</strong>（捕まえる、捕獲する）、
	<strong>gibe</strong>（与える）。ただし、特定の<em>他動詞</em>は、直接目的語を省略することがあります。：<strong>doxo</strong>（読む）、<strong>yam</strong>（食べる）、<strong>lala</strong>（歌う）など。
</p>
<p>一部の<em>他動詞</em>は、ほとんどの場合、直接目的語なしで使用されます。：<strong>somno</strong>（眠る）、
	<strong>haha</strong>（笑う）、<strong>pawbu</strong>（走る）、<strong>fley</strong>（飛ぶ）など。</p>
<p><strong>Mi le somno (lungo somno).</strong><br /> 私は（長い間）眠りました。</p>
<p><strong>Yu le haha (sotipul haha).</strong><br /> あなたは（大声で）笑いました。</p>
<p>接尾辞<strong><a
			href="./inharelexi.html#xafefikso_-gi">-gi</a></strong>は、<em>他動詞</em>に適用して、_（直接目的語）を［語根動詞］させる、（直接目的語）を［語根動詞］にする_という意味を持たせることができます。
</p>
<p><strong>Kam yu fleygi hawanavi?</strong><br /> あなたは航空機を操縦しますか？</p>
<p><strong>Mi xa sampogi bwaw fe axam.</strong><br /> 私は夕方に犬を散歩させます。</p>
<p><strong>Payaco le hahagi mi.</strong><br /> ピエロは私を笑わせました。</p>
<h4>自動詞</h4>
<p><em>自動詞</em>は直接目的語を取りません。：<strong>idi</strong>（行く）、<strong>konduta</strong>
	（振る舞う）、<strong>loka</strong>（位置する）、<strong>side</strong>（座っている）、<strong>garaku</strong>（溺れる、
	沈む）。<em>自動詞</em>は、直接目的語が存在する場合、<strong><a
			href="./inharelexi.html#xafefikso_-gi">-gi</a></strong>を任意で使用して<em>他動詞</em>に変えることができます。</p>
<ul>
	<li><strong>garaku</strong> - 溺れる、沈む<br />
		<strong>garaku(gi)</strong> - （を）溺れさせる、沈める
	</li>
</ul>
<p><strong>Navikef le garaku.</strong><br /> 船長は溺れました。</p>
<p><strong>Navikef le garaku(gi) navi.</strong><br /> 船長は船を沈めました。</p>
<p>ただし、語の派生では、<strong><a href="./inharelexi.html#xafefikso_-gi">-gi</a></strong>は必須です。
	<strong>garaku</strong>と接尾辞<strong><a href="./inharelexi.html#xafefikso_-ne">-ne</a></strong>から派生した次の形容詞を比較してください。
</p>
<p><strong>garakune navikef</strong> - 溺れている船長</p>
<p><strong>garakugine navikef</strong>- （船を）沈めている船長</p>
<h4>両用動詞</h4>
<p>グロバサの<a
		href="https://en.wikipedia.org/wiki/Ambitransitive_verb#Patientive"><em>両用動詞</em></a>は、自動詞の意味の主語と他動詞の意味の直接目的語の両方が、動詞の同じ動作/状態を経験する動詞です。これらの動詞の自動詞の意味は、任意で<strong><a
			href="./inharelexi.html#xafefikso_-cu">-cu</a></strong>を適用することができ、他動詞の意味は任意で<strong><a
			href="./inharelexi.html#xafefikso_-gi">-gi</a></strong>を適用することができます。</p>
<p><em>両用動詞</em>には、<em>感情動詞</em>、<em>状態動詞</em>、<em>無動作主動詞</em>、<em>位置/場所または移動の動詞</em>の4つのサブカテゴリーがあります。</p>
<p><em>感情動詞</em></p>
<p>感情を表す<em>両用動詞</em>は、<em>［語根名詞］を感じる、または［語根名詞］を感じさせる</em>という意味です。
	自動詞の意味は、コピュラと形容詞のフレーズとしても表現できることに注意してください。：<strong>sen</strong> <em>[語根 名詞]</em><strong><a
			href="./inharelexi.html#xafefikso_-do">-do</a></strong>。</p>
<ul>
	<li><strong>interes(cu)</strong> - 興味がある/感じる（<em>興味を感じる</em>）<br />
		<strong>interes(gi)</strong> - 興味を持たせる（<em>興味を感じさせる</em>）
	</li>
</ul>
<p><strong>Mi interes(cu) tem basalogi.</strong> = <strong>Mi sen interesdo tem basalogi.</strong><br /> 私は言語学に興味があります。
</p>
<p><strong>Basalogi interes(gi) mi.</strong><br /> 言語学は私に興味を持たせます。</p>
<ul>
	<li><strong>pilo(cu)</strong> - 疲れている/感じる（<em>疲労を感じる</em>）<br />
		<strong>pilo(gi)</strong> - 疲れさせる（<em>疲労を感じさせる</em>）
	</li>
</ul>
<p><strong>Te pilo(cu).</strong> = <strong>Te sen pilodo.</strong><br /> 彼女は疲れています/疲れていると感じています。</p>
<p><strong>Tesu ergo pilo(gi) te.</strong><br /> 彼女の仕事は彼女を疲れさせます。</p>
<p>接頭辞<strong>xor-</strong>は、<em>感情動詞</em>で使用して、次の違いを表現できます。</p>
<p><strong>Mi le interes tem basalogi lefe multi nyan.</strong><br /> 私は何年も前に言語学に興味がありました。</p>
<p><strong>Te le pilo dur na ergo.</strong><br /> 彼女は仕事中に疲れていました。</p>
<p>対して：</p>
<p><strong>Mi le xorinteres tem basalogi lefe multi nyan.</strong><br /> 私は何年も前に言語学に興味を持つようになりました。</p>
<p><strong>Te le xorpilo dur na ergo.</strong><br /> 彼女は仕事中に疲れてきました/疲れを感じるようになりました。</p>
<p><em>状態動詞</em></p>
<p><em>状態の両用動詞</em>は、<em>感情動詞</em>に似ています。これらは、<em>状態の名詞</em>であり、<em>［名詞の語根］を持つ、または［名詞の語根］を持たせる</em>という意味の<em>両用動詞</em>として使用できます。
	自動詞の意味は、コピュラと形容詞のフレーズとしても表現できることに注意してください。：<strong>sen</strong>
	<em>[語根名詞]</em><strong><a href="./inharelexi.html#xafefikso_-pul">-pul</a></strong>。
</p>
<ul>
	<li><strong>termo(cu)</strong> - 暑い/暖かい（<em>熱を持っている</em>）<br />
		<strong>termo(gi)</strong> - 温める（<em>熱を持たせる</em>）
	</li>
</ul>
<p><strong>Misu kafe no haji termo.</strong> = <strong>Misu kafe no haji sen termopul.</strong><br /> 私のコーヒーはもう熱くありません。
</p>
<p><strong>Kam yu le termo banyo-kamer?</strong><br /> あなたはバスルームを暖めましたか？</p>
<ul>
	<li><strong>cinon(cu)</strong> - 知的である（<em>知性を持っている</em>）<br />
		<strong>cinon(gi)</strong> - 知的にする（<em>知性を持たせる</em>）
	</li>
</ul>
<p><strong>Syensiyen cinon.</strong> = <strong>Syensiyen sen cinonpul.</strong><br /> その科学者は知的です。</p>
<p><strong>Eskol le cinon te.</strong><br /> 学校は彼を賢くしました。</p>
<ul>
	<li><strong>talento(cu)</strong> - 才能がある（<em>才能を持っている</em>）<br />
		<strong>talento(gi)</strong> - 才能を持たせる（<em>才能を持たせる</em>）
	</li>
</ul>
<p><strong>Lalayen talento.</strong> = <strong>Lalayen sen talentopul.</strong><br /> その歌手は才能があります。</p>
<p><strong>Patre le talento lalayen.</strong><br /> 父親は歌手に才能を持たせました。</p>
<p>接頭辞<strong>xor-</strong>は、<em>状態動詞</em>で使用して、次の違いを表現できます。</p>
<p><strong>Jaledin le termo.</strong> = <strong>Jaledin le sen termopul.</strong><br /> 昨日は暖かかったです。</p>
<p><strong>To le xortermo fe midinuru.</strong> = <strong>To le xorsen/sencu termopul fe midinuru.</strong><br />
	正午に暖かくなりました。</p>
<p><em>無動作主動詞</em></p>
<p><em>無動作主の両用動詞</em>の自動詞の意味では、動作は主語が行うものではなく、主語に起こるものです。</p>
<ul>
	<li><strong>kasiru(cu)</strong> - 壊れる（<em>壊れた状態になる</em>）<br />
		<strong>kasiru(gi)</strong> - 壊す（<em>壊す原因となる</em>）
	</li>
</ul>
<p><strong>Janela le kasiru(cu).</strong><br /> 窓が壊れました。</p>
<p><strong>Mi le kasiru(gi) janela.</strong><br /> 私は窓を壊しました。</p>
<ul>
	<li><strong>boyle(cu)</strong> - 沸騰する（<em>沸騰した状態になる</em>）<br />
		<strong>boyle(gi)</strong> - 沸騰させる（<em>沸騰させる原因となる</em>）
	</li>
</ul>
<p><strong>Sui le boyle(cu).</strong><br /> 水が沸騰しました。</p>
<p><strong>Te le boyle(gi) sui.</strong><br /> 彼女は水を沸騰させました。</p>
<ul>
	<li><strong>fini(cu)</strong> - 終わる（<em>終わりに来る</em>）<br />
		<strong>fini(gi)</strong> - 終わらせる、終了させる（<em>終わりに来させる</em>）
	</li>
</ul>
<p><strong>Jange le fini(cu).</strong><br /> 戦争が終わりました。</p>
<p><strong>Ete le fini(gi) jange.</strong><br /> 彼らは戦争を終わらせました。</p>
<p><em>位置/場所または移動の動詞</em></p>
<p><em>位置/場所または移動の両用動詞</em>の自動詞の意味では、主語は動作主と被動作主の両方です。</p>
<ul>
	<li><strong>esto(cu)</strong> - 止まる（<em>停止状態になる</em>）<br />
		<strong>esto(gi)</strong> - 止める（<em>停止状態にする、または停止させる</em>）
	</li>
</ul>
<p><strong>Am esto(cu)!</strong><br /> 止まれ！</p>
<p><strong>Am esto(gi) mobil!</strong><br /> 車を止めろ！</p>
<ul>
	<li><strong>harka(cu)</strong> - 動く（<em>動きをする</em>）<br />
		<strong>harka(gi)</strong> - 動かす（<em>動かす原因となる</em>）
	</li>
</ul>
<p><strong>Am no harka(cu)!</strong><br /> 動くな！</p>
<p><strong>Mi le harka(gi) yusu kursi.</strong><br /> 私はあなたの椅子を動かしました。</p>
<ul>
	<li><strong>buka(cu)</strong> - 開く（<em>開いた状態になる</em>）<br />
		<strong>buka(gi)</strong> - 開ける（<em>開いた状態にする</em>）
	</li>
</ul>
<p><strong>Dwer le buka(cu).</strong><br /> ドアが開きました。</p>
<p><strong>Mi le buka(gi) dwer.</strong><br /> 私はドアを開けました。</p>
<p>語の派生では、<em>両用動詞</em>は、<strong><a
			href="./inharelexi.html#xafefikso_-gi">-gi</a></strong>を必要とせずに<em>他動詞</em>として機能します。ただし、語の派生で動詞の<em>自動詞</em>の意味が必要な場合は、<strong><a
			href="./inharelexi.html#xafefikso_-cu">-cu</a></strong>を適用する必要があります。<strong>buka</strong>と接尾辞<strong><a
			href="./inharelexi.html#xafefikso_-ne">-ne</a></strong>から派生した次の形容詞を比較してください。</p>
<p><strong>bukane merasem</strong> - 開会式</p>
<p><strong>bukacune dwer</strong> - 開くドア</p>
<h2>形容詞/副詞</h2>
<p>グロバサでは、形容詞と<em>動詞を修飾する副詞</em>は同じ形をしています。</p>
<ul>
	<li><strong>bon</strong> - 良い、上手く</li>
	<li><strong>velosi</strong> - 速い、迅速に</li>
	<li><strong>multi</strong> - 多い、多く</li>
</ul>
<p>形容詞/副詞は、修飾する名詞/動詞の前に置かれます。</p>
<p><strong>Hinto sen <em>bon yam</em>.</strong><br /> これは<em>良い食事</em>です。</p>
<p><strong>Bebe <em>bon yam</em>.</strong><br /> 赤ちゃんは<em>上手に食べます</em>。</p>
<p><strong>Uma <em>velosi pawbu</em>.</strong><br /> 馬は<em>速く走ります</em>。</p>
<p>あるいは、副詞は動詞の後に出現する場合がありますが、その場合は、直接目的語と間接目的語(存在する場合)の後に続きます: 主語 - 動詞 - (直接目的語と間接目的語) - 副詞。</p>
<p><strong>Bebe yam <em>bon</em>.</strong><br /> 赤ちゃんは<em>上手く</em>食べます。</p>
<p><strong>Bwaw glu sui <em>velosi</em>.</strong><br /> 犬は水を<em>速く</em>飲みます。</p>
<p>副詞は文頭に移動することもできますが、その場合は文節を文の残りの部分から分離するように、コンマで明確な一時停止を入れる必要があります。一時停止がない場合、形容詞/副詞は主語を修飾するものとして誤って解釈される可能性があります。</p>
<p><strong><em>Velosi</em>, bwaw glu sui.</strong><br />
	<em>速く</em>、犬は水を飲みます。
</p>
<p><strong><em>Unyum</em>, te le idi cel banko.</strong><br />
	<em>最初に</em>、彼女は銀行に行きました。
</p>
<h3>文頭の句における形容詞/副詞</h3>
<p>以下は、文頭の句でよく使用される形容詞/副詞で、文の残りの部分の前に明確な一時停止が続きます。</p>
<ul>
	<li><strong>Ripul,</strong> - 再び</li>
	<li><strong>Ible,</strong> - 多分、おそらく</li>
	<li><strong>Maxpul,</strong> - さらに、そのうえ</li>
	<li><strong>Pia,</strong> - また</li>
	<li><strong>Abruto,</strong> - 突然</li>
	<li><strong>Total,</strong> - 絶対に、完全に</li>
	<li><strong>Yakin,</strong> - 確かに</li>
	<li><strong>Ideal,</strong> - 理想的に</li>
	<li><strong>Mimbay,</strong> - 明らかに、もちろん</li>
	<li><strong>Mingu,</strong> - 明らかに、明白に</li>
	<li><strong>Sipul,</strong> - 実際</li>
	<li><strong>Fori,</strong> - 直ちに</li>
	<li><strong>Pimpan,</strong> - しばしば、たびたび</li>
	<li><strong>Sati,</strong> - 本当に</li>
	<li><strong>Umumi,</strong> - 一般的に</li>
	<li><strong>Nerleli,</strong> - 最近</li>
	<li><strong>Telileli,</strong> - ずっと前に</li>
	<li><strong>Nerxali,</strong> - まもなく</li>
	<li><strong>Telixali,</strong> - 長い間</li>
</ul>
<h3>形容詞/副詞の比較</h3>
<p>形容詞/副詞の比較は、<strong>maxmo</strong> (<em>より～、～い</em>)、<strong>minmo</strong> (<em>より少ない</em>)、<strong>denmo</strong>
	(<em>～と同じくらい</em>)、<strong>kom</strong> (<em>として、より</em>)という言葉を使って、以下のように表現されます。</p>
<ul>
	<li><strong>maxmo kimapul kom...</strong> - ...よりも高価</li>
	<li><strong>minmo kimapul kom...</strong> - ...よりも安価</li>
	<li><strong>denmo kimapul kom...</strong> - ...と同じくらい高価</li>
</ul>
<p><em>最も～(～est)</em>と<em>最も少ない</em>を表現するために、グロバサは <strong>maxim... te/to</strong> と <strong>minim... te/to</strong>
	を使用します。<strong>of</strong> という言葉は、<em>～の中から</em>または<em>～(の)から</em>を意味します。名詞句は常に名詞または代名詞で終わる必要があるため、代名詞
	<strong>te</strong>/<strong>to</strong> は形容詞の直後に続く必要があることに注意してください。 <a
		href="./jumlemonli-estrutur.html#pornamelexi_in_namelexili_jumlemon">名詞句</a>を参照してください。</p>
<ul>
	<li><strong>maxim juni te (of misu bete)</strong> - (私の子供の中で)最も若い</li>
	<li><strong>minim kimapul to (of yusu mobil)</strong> - (あなたの車の中で)最も安価</li>
</ul>
<p><em>...すればするほど、...になる</em>を表現するために、グロバサは <strong>folki maxmo/minmo, maxmo/minmo</strong> を使用します。</p>
<ul>
	<li><strong>folki (to sen) maxmo neo, (to sen) maxmo bon</strong> - 新しいほど、良い</li>
</ul>
<h2>動詞/形容詞-副詞の語</h2>
<p>名詞/動詞の語と形容詞/副詞の語に加えて、グロバサには第3の種類の語があります: <em>動詞/形容詞-副詞の語</em>。助動詞のみがこの種類に属し、それは <strong>abil</strong>,
	<strong>musi</strong>, <strong>ingay</strong> の3語のみで構成されます。</p>
<ul>
	<li><strong>abil</strong>: (動詞) できる、～できる; (形容詞/副詞) 有能な、～できる</li>
	<li><strong>musi</strong>: (動詞) しなければならない; (形容詞/副詞) ～しなければならない、～する必要がある</li>
	<li><strong>ingay</strong>: (動詞) すべき、～するはず; (形容詞/副詞) ～すべき、～するはず</li>
</ul>
<h2>一般的な接辞 <span id="pimpan_fikso"></span></h2>
<h3>名詞接尾辞 -ya <span id="xafefikso_-ya"></span></h3>
<p>接尾辞 <strong>-ya</strong> にはさまざまな便利な機能があり、いくつかの英語の接尾辞に相当します:
	<em>-ity</em>、<em>-ness</em>、<em>-dom</em>、<em>-hood</em>、<em>-ship</em>。</p>
<ol>
	<li>抽象名詞は、形容詞/副詞に <strong>-ya</strong> を追加することで派生されます。</li>
</ol>
<ul>
	<li>
		<p><strong>real</strong> - 本当の (<em>形容詞</em>)<br />
			<strong>realya</strong> - 現実 (<em>名詞</em>)
		</p>
	</li>
	<li>
		<p><strong>bimar</strong> - 病気の (<em>形容詞</em>)<br />
			<strong>bimarya</strong> - 病気 (<em>名詞</em>)
		</p>
	</li>
	<li>
		<p><strong>huru</strong> - 自由な (<em>形容詞</em>)<br />
			<strong>huruya</strong> - 自由 (<em>名詞</em>)
		</p>
	</li>
	<li>
		<p><strong>solo</strong> - 一人で (<em>形容詞</em>)<br />
			<strong>soloya</strong> - 孤独 (<em>名詞</em>)
		</p>
	</li>
</ul>
<ol start="2">
	<li>接尾辞 <strong>-ya</strong> は、さまざまな具体的および可算名詞から抽象的および不可算名詞を派生するために使用されます。</li>
</ol>
<ul>
	<li><strong>poema</strong> - 詩 (<em>具体的な名詞</em>) </li>
	<li><strong>poemaya</strong> - 詩 (<em>抽象名詞</em>)</li>
</ul>
<p>接尾辞 <strong>-ya</strong> は、関係を示す名詞に付加されると、<em>-hood</em> または <em>-ship</em> を意味します。</p>
<ul>
	<li>
		<p><strong>matre</strong> - 母 (<em>具体的な名詞</em>)<br />
			<strong>matreya</strong> - 母性 (<em>抽象名詞</em>)
		</p>
	</li>
	<li>
		<p><strong>patre</strong> - 父 (<em>具体的な名詞</em>)<br />
			<strong>patreya</strong> - 父性 (<em>抽象名詞</em>)
		</p>
	</li>
	<li>
		<p><strong>doste</strong> - 友人 (<em>具体的な名詞</em>)<br />
			<strong>dosteya</strong> - 友情 (<em>抽象名詞</em>)
		</p>
	</li>
</ul>
<p>場合によっては、具体的な名詞または可算名詞が動詞として使用され、抽象名詞または不可算名詞が <strong>-ya</strong> を使用して派生され、動詞に対応する機能があります。</p>
<ul>
	<li>
		<p><strong>imaje</strong> - イメージ/写真 (<em>具体的な名詞</em>), 想像する/描く (<em>動詞</em>)<br />
			<strong>imajeya</strong> - 想像力 (<em>抽象名詞</em>)
		</p>
	</li>
	<li>
		<p><strong>turi</strong> - 旅行 (<em>可算名詞</em>), 旅行する (<em>動詞</em>)<br />
			<strong>turiya</strong> - 観光 (<em>不可算名詞</em>)
		</p>
	</li>
</ul>
<p>同様に、五感に関連する体の部位は関連する行動(<em>動詞</em>)を示しますが、<strong>-ya</strong> は抽象名詞を派生するために使用されます。</p>
<ul>
	<li>
		<p><strong>oko</strong> - 目 (<em>具体的な名詞</em>), 見る, 見つめる (<em>動詞</em>)<br />
			<strong>okoya</strong> - 視力, 見解 <em>または</em> 視覚 (<em>抽象名詞</em>)
		</p>
	</li>
	<li>
		<p><strong>ore</strong> - 耳 (<em>具体的な名詞</em>), 聞く (<em>動詞</em>)<br />
			<strong>oreya</strong> - 聴覚 <em>または</em> 聴覚 (<em>抽象名詞</em>)
		</p>
	</li>
	<li>
		<p><strong>nasa</strong> - 鼻 (<em>具体的な名詞</em>), 匂いを嗅ぐ (<em>動詞</em>)<br />
			<strong>nasaya</strong> - 匂い <em>または</em> 嗅覚 (<em>抽象名詞</em>)
		</p>
	</li>
	<li>
		<p><strong>xeto</strong> - 舌 (<em>具体的な名詞</em>), 味をみる (<em>動詞</em>)<br />
			<strong>xetoya</strong> - 味 <em>または</em> 味覚 (<em>抽象名詞</em>)
		</p>
	</li>
	<li>
		<p><strong>pifu</strong> - 皮膚 (<em>具体的な名詞</em>), 触れる (<em>動詞</em>)<br />
			<strong>pifuya</strong> - 感触 <em>または</em> 触覚 (<em>抽象名詞</em>)
		</p>
	</li>
</ul>
<ol start="3">
	<li>
		<p>前置詞は、接尾辞 <strong>-ya</strong> を使用して名詞/動詞に変わります。 <a
				href="./jumleli-estrutur.html#plasilexili_falelexi">前置詞動詞</a>を参照してください。</p>
	</li>
	<li>
		<p>接尾辞 <strong>-ya</strong> は、他の機能語を名詞に変えるためにも使用されます。 <a href="./gramatilexi.html">機能語</a>を参照してください。</p>
	</li>
</ol>
<p><em><strong>-ya</strong> の語源: ヒンディー語 (सत्य "satya" - 真実)、スペイン語 (alegría - 喜び)</em></p>
<h3>接頭辞 du- <span id="lefefikso_du-"></span></h3>
<p>グロバサは、接頭辞 <strong>du-</strong> を使用して動名詞を表現します。</p>
<ul>
	<li><strong>dudanse</strong> - (～する行為としての)ダンス</li>
	<li><strong>dulala</strong> - (～する行為としての)歌</li>
</ul>
<p>接頭辞 <strong>du-</strong> は、<em>継続的/習慣的な動詞の相</em>にも使用されます。 <a href="./falelexili-morfo.html">動詞の形</a>を参照してください。</p>
<p><em>接頭辞 <strong>du-</strong> は <strong>dure</strong> (継続) から短縮されています。<br /> <strong>dure</strong> の語源:
		英語、フランス語、ドイツ語、スペイン語</em></p>
<h3>名詞/動詞接尾辞 -gi <span id="xafefikso_-gi"></span></h3>
<p>接尾辞 <strong>-gi</strong> は、形容詞、名詞、動詞に適用できます。</p>
<h4>形容詞</h4>
<p>接尾辞 <strong>-gi</strong> は、形容詞を他動詞に変えます。</p>
<ul>
	<li>
		<p><strong>bala</strong> - 強い<br />
			<strong>balagi</strong> - 強化する
		</p>
	</li>
	<li>
		<p><strong>pul</strong> - いっぱいの<br />
			<strong>pulgi</strong> - 満たす
		</p>
	</li>
	<li>
		<p><strong>mor</strong> - 死んだ<br />
			<strong>morgi</strong> - 殺す
		</p>
	</li>
</ul>
<h4>名詞</h4>
<p>接尾辞 <strong>-gi</strong> は、名詞に追加されると、<em>～にする (～になる)</em> を意味します。</p>
<ul>
	<li>
		<p><strong>zombi</strong> - ゾンビ<br />
			<strong>zombigi</strong> - ゾンビ化する
		</p>
	</li>
	<li>
		<p><strong>korbani</strong> - 被害者<br />
			<strong>korbanigi</strong> - 被害を与える
		</p>
	</li>
</ul>
<h4>動詞</h4>
<p>接尾辞 <strong>-gi</strong> は、自動詞と他動詞を使役動詞に変えるため、または上記の<a
		href="./inharelexi.html#falelexili_klase"><em>動詞のカテゴリー</em></a>で示されているように、両他動詞で任意に使用されます。</p>
<p><em>接尾辞 <strong>-gi</strong> は <strong>gibe</strong>（与える）から切り捨てられています。<br /> <strong>gibe</strong> の語源：英語 (give)、ドイツ語
		(geben, gibt)、マンダリン (给 “gěi”)</em></p>
<h3>名詞・動詞の接尾辞 -cu <span id="xafefikso_-cu"></span></h3>
<p>接尾辞 <strong>-cu</strong> は、上記の<a
		href="./inharelexi.html#falelexili_klase"><em>動詞のカテゴリー</em></a>で示されているように、動詞に加えて形容詞や名詞にも適用できます。</p>
<h4>形容詞</h4>
<p>接尾辞 <strong>-cu</strong>（なる・得る）は形容詞を自動詞に変えます。</p>
<ul>
	<li>
		<p><strong>roso</strong> - 赤い<br />
			<strong>rosocu</strong> - 赤くなる (<em>赤くなる</em>)
		</p>
	</li>
	<li>
		<p><strong>mor</strong> - 死んだ<br />
			<strong>morcu</strong> - 死ぬ (<em>死んだ状態になる</em>)
		</p>
	</li>
</ul>
<h4>名詞</h4>
<p>接尾辞 <strong>-cu</strong> は、名詞に追加されると、<em>～になる</em>ことを意味します。</p>
<ul>
	<li>
		<p><strong>zombi</strong> - ゾンビ<br />
			<strong>zombicu</strong> - ゾンビになる
		</p>
	</li>
	<li>
		<p><strong>ixu</strong> - (人間の)大人<br />
			<strong>ixucu</strong> - 大人になる、成人する
		</p>
	</li>
</ul>
<p><em>接尾辞 <strong>-cu</strong> は <strong>cudu</strong> (取る、獲得する) から短縮されています。<br /> <strong>cudu</strong> の語源: 中国語 (取得
		"qǔdé")、韓国語 (취득 “chwideug”)</em></p>
<h3>形容詞/副詞接尾辞 -li <span id="xafefikso_-li"></span></h3>
<p>グロバサでは、形容詞/副詞は、さまざまな接尾辞によって名詞から派生されます。 <a href="./lexikostrui.html">単語形成</a>の接尾辞の完全なリストを参照してください。最も一般的なものの1つは、接尾辞
	<strong>-li</strong> (<em>～の、～に関する</em>)です。</p>
<ul>
	<li>
		<p><strong>musika</strong> - 音楽<br />
			<strong>musikali</strong> - 音楽の、音楽的に
		</p>
	</li>
	<li>
		<p><strong>denta</strong> - 歯<br />
			<strong>dentali</strong> - 歯科の
		</p>
	</li>
	<li>
		<p><strong>dongu</strong> - 東<br />
			<strong>donguli</strong> - 東の
		</p>
	</li>
	<li>
		<p><strong>Franse</strong> - フランス<br />
			<strong>Franseli</strong> - フランスの
		</p>
	</li>
</ul>
<p>接尾辞 <strong>-li</strong> は、機能語から形容詞/副詞を派生させるためにも使用されます。 <a href="./gramatilexi.html">機能語</a>を参照してください。</p>
<p><em><strong>-li</strong> の語源: フランス語 (-el, -elle)、スペイン語 (-al)、英語 (-al, -ly)、ドイツ語 (-lich)、ロシア語 (-ельный “-elni”,
		-альный “-alni”)、トルコ語 (-li)</em></p>
<h3>形容詞/副詞接尾辞 -pul <span id="xafefikso_-pul"></span></h3>
<p>単語 <strong>pul</strong> は、<em>いっぱい</em> を意味します。ただし、接尾辞として <strong>-pul</strong> は、<em>十分以上</em> を意味します。</p>
<ul>
	<li>
		<p><strong>humor</strong> - ユーモア<br />
			<strong>humorpul</strong> - ユーモラスな、面白い
		</p>
	</li>
	<li>
		<p><strong>hatari</strong> - 危険<br />
			<strong>hataripul</strong> - 危険な
		</p>
	</li>
</ul>
<p><em><strong>pul</strong> の語源: 英語 (full)、ヒンディー語 (पूर्ण “purn”)、ロシア語 (полный “poln-”)</em></p>
<h3>能動態形容詞: 接尾辞 -ne <span id="xafefikso_-ne"></span></h3>
<p>接尾辞 <strong>-ne</strong> は、<em>能動的な状態またはプロセスにある</em>ことを意味し、グロバサで<em>能動態形容詞</em>として知られているものを派生させるために使用されます。</p>
<p><em>能動態形容詞</em>は、ほとんどの場合、英語の<em>現在分詞</em>(<em>-ing</em>で終わる形容詞)に相当します。ただし、英語とは異なり、<em>能動態形容詞</em>は<em>進行形動詞</em>(<em>I
		am sleeping</em>、<em>She is dancing</em>など)を生成するためには使用されません。代わりに、それらは形容詞としてのみ機能します。</p>
<ul>
	<li>
		<p><strong>somno</strong> - 眠る<br />
			<strong><em>somnone</em> meliyen</strong> - <em>眠っている</em> 美女
		</p>
	</li>
	<li>
		<p><strong>anda</strong> - 歩く<br />
			<strong><em>andane</em> moryen</strong> - <em>歩いている</em> 死者
		</p>
	</li>
	<li>
		<p><strong>danse</strong> - ダンスする<br />
			<strong><em>dansene</em> uma</strong> - <em>踊っている</em> 馬
		</p>
	</li>
	<li>
		<p><strong>interes</strong> - 興味<br />
			<strong><em>interesne</em> kitabu</strong> - <em>興味深い</em> 本
		</p>
	</li>
	<li>
		<p><strong>amusa</strong> - 面白がらせる、楽しい<br />
			<strong><em>amusane</em> filme</strong> - <em>面白い/楽しい</em> 映画
		</p>
	</li>
</ul>
<p><em><strong>-ne</strong> の語源: 英語 (-ing)、フランス語 (-ant)、スペイン語 (-ando)、ドイツ語 (-en, -ende)、ロシア語 (-ный “-ny”)、トルコ語 (-en,
		-an)</em></p>
<h3>文頭の能動態形容詞</h3>
<p>文頭に現れる能動態形容詞は、代わりに不定形動詞を使った前置詞句として表現できます。</p>
<p><strong><em>Doxone,</em> nini le xorsomno.</strong><br />
	<em>読んでいると、</em> 子供は眠ってしまった。
</p>
<p><em>または</em></p>
<p><strong><em>Fe na doxo,</em> nini le xorsomno.</strong><br />
	<em>読んでいると、</em> 子供は眠ってしまった。<br />
	<em>または</em><br />
	<strong><em>Dur na doxo,</em> nini le xorsomno.</strong><br />
	<em>読んでいる間、</em> 子供は眠ってしまった。
</p>
<p>この構造は、特にフレーズに直接目的語が含まれる場合に便利です。英語の<em>現在分詞</em>とは異なり、グロバサの<em>能動態形容詞</em>は動詞として機能できないためです。</p>
<p><strong><em>Dur na doxo sesu preferido kitabu,</em> nini le xorsomno.</strong><br />
	<em>彼のお気に入りの本を読んでいる間、</em> 子供は眠ってしまった。
</p>
<p>当然のことながら、これらのフレーズは、前置詞句とは対照的に、完全な節としても表現できます。</p>
<p><strong><em>Durki te le doxo (sesu preferido kitabu),</em> nini le xorsomno.</strong><br />
	<em>彼が(彼のお気に入りの本を)読んでいる間、</em> 子供は眠ってしまった。
</p>
<h3>受動態形容詞</h3>
<p><em>能動態形容詞</em>は、受動態の接頭辞 <strong>be-</strong>
	を追加して、グロバサで<em>受動態能動態形容詞</em>(または短く<em>受動態形容詞</em>)として知られているものを派生させることで、受動態にすることができます。英語には<em>受動態形容詞</em>に相当するものがありませんが、英語の<em>現在分詞</em>の正確な受動態として理解するのが最適です。
</p>
<ul>
	<li><strong>belalane melodi</strong> - 歌われるまたは歌われているメロディー</li>
	<li><strong>belubine doste</strong> - 愛する友人または愛されている友人</li>
</ul>
<h3>非能動態形容詞: 接尾辞 -do <span id="xafefikso_-do"></span></h3>
<p>接尾辞 <strong>-do</strong>
	は、<em>非能動的な状態にある</em>ことを意味します。この接尾辞を持つ単語は、グロバサでは<em>非能動態形容詞</em>として知られており、通常、英語の<em>過去分詞</em>として翻訳されます。ただし、英語とは異なり、<em>非能動態形容詞</em>は<em>完了</em>または<em>受動態動詞</em>(<em>I
		have worked</em>、<em>It was/got stolen</em>など)を生成するためには使用されません。代わりに、形容詞としてのみ機能します。</p>
<p>厳密に言えば、接尾辞 <strong>-do</strong> は、名詞/動詞の<em>名詞</em>の側面に追加されることに注意する価値があります。このため、<strong>-do</strong>
	は他動詞、自動詞、または両用動詞の名詞/動詞に追加できます。</p>
<h4>他動詞の場合</h4>
<ul>
	<li><strong>hajado ergo</strong> - 必要な仕事 (<em>必要性の状態にある</em>)</li>
	<li><strong>bujodo morgiyen</strong> - 捕獲された殺人者 (<em>捕獲の状態にある</em>)</li>
</ul>
<h4>自動詞の場合</h4>
<ul>
	<li><strong>Uncudo Nasyonlari</strong> - 国際連合 (<em>連合の状態にある</em>)</li>
	<li><strong>awcudo fleytora</strong> - 行方不明の飛行機 (<em>行方不明の状態にある</em>)</li>
</ul>
<h4>両用動詞の場合</h4>
<ul>
	<li><strong>kasirudo janela</strong> - 壊れた窓 (<em>破損の状態にある</em>)</li>
	<li><strong>klosido dwer</strong> - 閉じたドア (<em>閉鎖の状態にある</em>)</li>
</ul>
<p><em><strong>-do</strong> の語源: 英語 (-ed)、スペイン語 (-ado, -ido)</em></p>
<h3>副詞接尾辞 -mo <span id="xafefikso_-mo"></span></h3>
<p><em>形容詞/副詞を修飾する副詞</em>として知られる、他の形容詞/副詞を修飾する形容詞/副詞は、接尾辞 <strong>-mo</strong> を追加します。次のフレーズのペアを比較してください。</p>
<ul>
	<li>
		<p><strong>perfeto blue oko</strong> - 完璧な青い目 (<em>完璧な青い目</em>)<br />
			<strong>perfetomo blue oko</strong> - 完璧に青い目 (<em>完璧に青い目</em>)
		</p>
	</li>
	<li>
		<p><strong>naturali syahe tofa</strong> - 自然な黒髪 (<em>かつらではない</em>)<br />
			<strong>naturalimo syahe tofa</strong> - 自然に黒い髪 (<em>染めていない</em>)
		</p>
	</li>
	<li>
		<p><strong>sotikal doxone nini</strong> - 静かに読書する子供<br />
			<strong>sotikalmo doxone nini</strong> - 静かに読書する子供
		</p>
	</li>
</ul>
